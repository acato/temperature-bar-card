# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-02-20

### Added
- User documentation suite in `docs/` folder
  - Installation guide (HACS custom repo setup, manual install, updating)
  - Configuration reference (all options, color/threshold system explained)
  - Examples (10 copy-paste YAML recipes for common setups)
  - Troubleshooting guide (common problems and fixes)

### Changed
- Rewrote README.md as a user-facing landing page with quick start and doc links
- Rewrote info.md for a cleaner HACS storefront page with links to full docs

## [1.1.0] - 2024-01-15

### Added
- Fahrenheit and Celsius unit support with automatic conversion
- Customizable temperature thresholds and colors
- Optional thermometer icons with customization
- Custom text color configuration
- Compact bar design with 30px height and rounded corners
- Better responsive design for mobile devices

### Changed
- Default unit is now Fahrenheit
- Smaller, more compact bar design
- Improved text layout and spacing
- Enhanced color gradient system

### Fixed
- Color gradient rendering issues
- Icon sizing and positioning
- Text overflow handling
- Layout alignment problems

## [1.0.0] - 2024-01-01

### Added
- Initial release
- Temperature-based gradient backgrounds
- Basic temperature display
- Home Assistant theme integration
- HACS compatibility
